A browser-based pixel city builder game.

Available at: https://tom-draper.github.io/city-builder/



Run <code>index.html</code> in a browser to run locally.

### Features
- [X] Animated water
- [X] Travelling cars
- [X] Meandering animals
- [X] Fishing boats
- [ ] Airports and planes
- [ ] Time + changing seasons
- [ ] Points system
